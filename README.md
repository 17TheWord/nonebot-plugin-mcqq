# 功能

- 推送消息列表
  - 服务器 -> QQ
    - 加入 / 离开 服务器消息
    - 玩家聊天信息
  - QQ -> 服务器
    - 群员聊天文本
    - 图片、视频等内容转换为 `[图片]`、`[视频]`

- 特殊消息支持
  - 群聊
    - [x] @ 消息
    - [ ] 回复消息
  - 频道
    - [x] @ 消息
    - [ ] 回复消息
  - 未支持的消息已被替换，如： `[图片]`、 `[视频]` 等等
